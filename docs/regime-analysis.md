## 읽는 법
- **KR(상폐보정)을 신뢰하라:** 200일선 **위**에서 배치하면 250d 손실이 −2.8% vs 아래 −16.6% → 지수 하락국면이면 비중 축소·대기, 상승국면에 적극(점수=무엇, 레짐=언제).
- ★ **US는 정반대로 나오지만 믿지 마라 — 생존편향 아티팩트.** US는 상폐 무료피드가 없어 *crash(200일선 아래)에 산 종목 중 살아남은 것만* 표본에 남아 'crash 매수가 더 좋다'는 착시를 만든다. 0으로 간 종목이 빠졌을 뿐. 실제로는 KR처럼 상승국면 배치가 옳다.
- 분기 표본이라 방향성으로 — 운영하며 forward 데이터로 재확인.
