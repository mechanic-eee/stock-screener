# 용어집 — 어원으로 이해하는 시스템 용어

> 형식: **용어 (원어)** — 어원/원뜻 → 투자에서의 뜻 → **우리 시스템에서의 실제 예시**.
> 매뉴얼·알림·성적표에 나오는 순서가 아니라, 자주 나오는 순서.

## 성적표에 나오는 것

**코호트 (cohort)** — 로마 군단의 보병대(같이 싸우는 무리) → 통계에서 "같은 시기에 시작해 함께 추적되는 집단". 의학에서 "같은 해에 태어난 사람들"을 추적할 때 쓰는 그 말.
→ 우리: **같은 날 워치리스트에 들어간 종목 묶음**. 5/25 코호트(점수만 100점, 대조군), 6/25 코호트(펀더 랭킹), 7/18 페이퍼 코호트(깔때기 통과 5픽). 왜 묶나: 산 날이 다르면 보유 기간이 달라서 수익률 직접 비교가 불공정 — 같은 날 출발한 묶음끼리 비교해야 공정하다.

**벤치마크 (benchmark)** — 측량사가 바위에 새기던 기준점 표시(bench mark) → "비교의 기준".
→ 우리: **KOSPI(한국 시장 평균)와 S&P500(미국 시장 평균)**. "**코호트 vs 벤치마크**" = 내 종목 묶음의 성적을 같은 기간 시장 평균과 나란히 놓는 것. 이게 왜 중요하냐면: "+2.9% 벌었다"만 보면 실력인지 시장이 다 오른 덕인지 구분이 안 된다. 실제 예: 6/25 코호트 +2.9%인데 같은 기간 KOSPI가 −23.6% → 시장보다 **26%p 잘한 것**(대단한 성적). 반대로 시장이 +30%인데 내가 +3%면 못한 것.

**엣지 (edge)** — 칼날/모서리 → 도박 용어 "하우스 엣지"(카지노가 손님보다 유리한 몇 %) → **통계적 우위**.
→ 우리: 백테스트로 확인된 우위 = 픽당 +1.3~3.8%p. 단 이건 "무조건 벌어준다"가 아니라 "같은 날 폭락주를 아무거나 산 사람보다 평균적으로 그만큼 낫다"는 **상대적** 우위다.

## 알림에 나오는 것

**레짐 (regime)** — 프랑스어 régime(통치 체제, 정권) → 시장의 "**국면/체제**". "정권이 바뀌면 규칙이 바뀐다"처럼, 시장도 상승 체제냐 하락 체제냐에 따라 같은 전략의 성적이 완전히 달라진다.
→ 우리: **지수(KOSPI/S&P)가 200일선 위면 상승 레짐, 아래면 하락 레짐**. 검증 결과가 극적이다: 200일선 위에서 산 폭락주는 250일 평균 −2.8%인데, 아래에서 사면 −16.6%. 그래서 규칙이 "하락 레짐에선 신규 진입 금지". 알림의 "시장:" 라인이 이것이고, 옆의 ±%는 200일선까지 남은 거리(현재 KR +22% = 여유, US +6.7% = 상대적으로 타이트).

**200일선 (200-day moving average)** — 최근 200거래일(약 10개월) 종가의 평균을 이은 선 → 장기 추세의 기준선. 주가가 이 위에 있으면 장기 상승 추세로 본다.

**하트비트 (heartbeat)** — 심장박동 → IT에서 "살아있다는 주기적 신호". 🩺 하트비트가 오면 감시 시스템이 정상 작동 중이라는 뜻. **안 오면 알림이 없는 게 아니라 감시가 죽은 것** — 이 구분이 존재 이유.

**thesis-break (시시스 브레이크)** — thesis(그리스어 "놓다"→명제, 논문) = 이 종목을 산 **논거**. thesis-break = 그 논거가 깨지는 사건(손절 이탈, 위험공시, 대규모 증자) → 청산 사유.

**ATR (Average True Range)** — 평균 진폭. 하루에 보통 얼마나 출렁이는지(고가−저가 개념의 14일 평균). ATR 5% = 하루 ±5%쯤 움직이는 종목.
→ 우리: **손절선 = 진입가 − 2.5×ATR**(정상 출렁임의 2.5배 밑으로 빠지면 정상 변동이 아니라 논거가 깨진 것) + ATR 8% 초과 종목은 후보 제외(복권 수준 변동성).

## 깔때기·기록에 나오는 것

**유니버스 (universe)** — 우주 → "전략이 대상으로 삼는 종목 전체". 우리 유니버스 = 5년 고가 대비 −50% 이상 폭락한 KR+US 종목(~1,200개).

**드로다운 (drawdown)** — draw down(끌어내리다) → **고점 대비 낙폭**. "낙폭 66%" = 5년 최고가에서 66% 빠진 상태. 우리 유니버스의 입장 자격.

**시드 (seed)** — 씨앗 → 워치리스트에 후보를 "심는" 것. 자동 시드 = 스크리너가 뽑아 심어준 초안.

**베토 (veto)** — 라틴어 "나는 금지한다"(로마 호민관의 거부권) → 게이트를 다 통과한 후보를 **사람 판단으로 탈락**시키는 것. 주 2건 상한(사람 직관이 이 유니버스에서 자주 틀리므로) + 베토한 종목도 계속 추적해 "베토가 실제로 옳았나"를 검증한다.

**트랜치 (tranche)** — 프랑스어 "조각, 슬라이스"(빵 한 조각) → 분할 매수의 한 조각. 1차 트랜치 50% 지금, 2차 트랜치 50%는 2~4주 뒤 — 하루에 다 사는 위험을 시간으로 나눈다.

**R-사이징 (R-sizing)** — R = Risk의 약자, **1회 트레이드에서 허용하는 최대 손실**(우리는 계좌의 1%). 수량 = R금액 ÷ (진입가−손절가). 손절가가 가까우면 많이 사고 멀면 조금 사서, 어떤 종목이든 "손절당하면 계좌의 1%만 잃게" 만드는 계산법.

**페이퍼 (paper trading)** — "종이 위" 거래 → 실제 돈 없이 규칙대로 기록만 하는 모의투자. 지금 5픽이 페이퍼(~9/12까지). 목적은 수익이 아니라 **파이프라인과 규칙 준수가 실제로 돌아가는지** 확인.

**게이트 (gate)** — 문/관문 → 통과 못 하면 탈락하는 하드 조건(펀더 결측·ATR·부도 이중확인·유동성). "게이트 통과"는 "안전하다"가 아니라 "탈락 사유를 못 찾았다"다.

## 검증 문서에 나오는 것 (깊이 안 가도 됨)

**IC (Information Coefficient, 정보계수)** — 점수 순위와 실제 수익 순위가 얼마나 일치하는가(상관계수). 0 = 점수가 동전던지기, 0.1이면 실전에서 꽤 유의미. 우리 합성점수는 KR 250d IC +0.15.

**t값 (t-statistic)** — "이 결과가 우연일 가능성"의 역수 느낌. |t| > 2면 우연이 아니라고 본다. 우리 펀더 신호 t4~7 = 우연일 가능성 극히 낮음. off_lows t−5.5 = "반등 확인 후 매수가 손해"라는 것도 우연이 아님.

**생존편향 (survivorship bias)** — 살아남은 것만 보이는 착시. 전사자는 말이 없다 — 상폐된 종목은 데이터에서 사라져서, 과거 데이터로 백테스트하면 성적이 실제보다 좋아 보인다. 우리는 KR 상폐 337종목을 복원해 보정했고, US 백테스트 절대수익(+18.5%)은 이 착시라서 안 믿는다.

**PIT (point-in-time, 그 시점 기준)** — 백테스트할 때 "그 날짜에 실제로 알 수 있었던 정보만" 쓰는 원칙. 예: 3월 실적은 5월에 공시되므로 4월 백테스트에 쓰면 미래를 훔쳐본 것(look-ahead). 우리 검증이 신뢰받는 핵심 이유.

**fail-open / fail-closed** — 고장 났을 때 열리느냐(통과시키느냐) 닫히느냐(차단하느냐). 금고문은 정전 시 잠겨야(fail-closed) 안전하다. 우리 감시도 "조회 실패 = 통과"였던 걸 전부 "실패 = 경고 표시"로 고쳤다(⚪ 미수행 표시가 그것).

**OOS (out-of-sample, 표본 밖)** — 규칙을 만들 때 안 쓴 새 데이터. 규칙을 만든 데이터로 규칙을 채점하면 자기 채점이라 무의미 — 지금 페이퍼 12주가 하는 일이 OOS 검증이다.
