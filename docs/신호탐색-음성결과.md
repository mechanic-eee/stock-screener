## 메타 교훈
- **음성결과의 연속 = 포화 신호.** 새 레버가 일관되게 엣지를 못 내면 기존 설정이 최적 근처 → 더 만지면 과적합. 멈추는 게 실력.
- **생존편향 주의:** US(상폐 누락) 결과의 *역전*(레짐·극단분위·debt)은 신호가 아니라 아티팩트. 신뢰는 KR(상폐보정)·횡단면 rank IC.
- **실제 엣지는 이미 안에 있다:** 드로다운 종곡선 + 부도회피(Altman·debt·capital_impairment) + 퀄리티(GP·Piotroski·op_margin) + 저변동성(ATR). 타이밍은 신호 아닌 *시장 200일선 레짐* 규칙.

## 그래도 가치 있을 미탐색 (큰 데이터 작업, 별도 세션)
- **내부자 매수 본검증** — probe는 음수였으나 표본 작음. 전 코호트(~50k Form4 fetch)로 본검증하면 결론 확정. 단 probe 방향이 음수라 우선순위 낮음.
- forward 추적 누적(track.py) — 가장 정직한 라이브 검증, 시간이 쌓아줌.
