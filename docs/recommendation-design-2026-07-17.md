# 추천 파이프라인 설계 — 점수에서 실전 매수 추천까지 (2026-07-17)

> **작성 경위:** 사용자 질문 "스코어링까지 했는데 실제 투자할 종목 추천은 어떻게?"에 대해
> 투자 전문가 에이전트가 설계 → 퀀트 적대적 검증 에이전트가 원문 문서 대조로 반박 → 종합.
> 인용 검증 8건 중 치명적 오인용 0건. 단 **하드 게이트의 숫자 파라미터는 전원 미검증 재량**으로
> 판정되어 아래에 전부 `[재량]` 표기했다. `[검증]`은 프로젝트 백테스트 문서에 근거가 있는 규칙.

## §0 핵심 진단 — 왜 점수 상위 ≠ 매수 추천인가

점수는 **횡단면 랭킹 신호**(사전확률)다. per-date IC t4~7은 "줄 세우면 위가 아래보다 낫다"이지 "위를 사면 돈 번다"가 아니다. 프로젝트 자체 수치로:

- 상위10 pooled 250d **−10.8%**(KR)인데 날짜중립 엣지는 **+1.3%p** → 랭킹은 "덜 나쁨"의 보장일 뿐, 절대수익은 레짐(200일선 위 −2.8% vs 아래 −16.6%)이 지배.
- SMCI 실사례: 점수 80으로 시드 → 대만 검찰·$7B 증자를 점수는 못 봄(뒤늦게 64.8 강등) → 공시·희석은 점수 입력에 없다. 사람 체크가 구조적으로 필요.
- base 점수는 100점 동률 무더기(5/25 시드 10종 전원 100점·전원 미큐레이션 폐기) → 점수는 후보 풀 축소기이지 최종 선택기가 아님.

**추천 = 점수(사전확률) × 레짐 게이트 × 악재 부재 확인 × 포트폴리오 제약 × 실행계획.** 현재 시스템은 양끝(점수, decide/position_size/monitor)만 있고 가운데가 암묵지다. 이 문서가 그 가운데를 명문화한다.

### 단계 5 — 사이징 `[검증]` (기존 도구)
- 손절 = 진입가 − 2.5×ATR (to_watchlist 자동초안). R = 계좌 1%, **첫 8주 0.5%**.

### 단계 6 — 포트폴리오 한도 `[재량]`
- 최대 동시 보유 6종목, 총 오픈리스크 Σ(진입−손절)×수량 ≤ 계좌 6% (6종목×R1%=6%로 정합 — 8종목이면 캡과 모순이라 6으로 통일), 전략 슬리브 ≤ 계좌 60%(pooled 음수 분기 다수 — 전략 전체가 1~2년 마이너스일 수 있음), 레짐 아래면 신규 R 0.

### 단계 7 — 보유·청산·리뷰
- 목표 보유 120~250d `[검증]` (KR 250d IC +0.151 t5.7; 60d 성과로 판단 금지).
- 청산: (a) ATR 손절 이탈 (b) thesis-break(위험공시·감사의견·희석 + **보유분이 시장 랭킹 하위 50%로 강등** `[재량]` — 검증 평결이 지적한 공백 보강) (c) 250d 도달 시 재평가: 상위 25% + 펀더 받침이면 연장 `[재량]`, 연장분은 태그해 자체 검증 데이터화(250d 초과는 전면 미검증 영역).
- 트레일링 스톱·이익실현 규칙 도입 안 함 `[검증]` (검증된 청산 신호 없음).
- 리듬: 일일 daily.ps1(기존) / 주간 깔때기 / 분기 리뷰 — **규칙 개정은 분기에만**.

## §3 Forward 검증 (3-arm, 목표 재정의)

| Arm | 무엇 | 상태 |
|---|---|---|
| A | base-only 시드 | 운영 중 |
| B | enrichment 상위 15 | 운영 중 ("점수만" 대조군) |
| C | 주간 깔때기 3~5픽 + 실행규칙 | 신설 (처치군) |

- **사전 등록**: 이 문서의 규칙·숫자를 버전 고정, 12주 변경 금지. 개정은 분기 리뷰에서만 + 코호트 버전 태그.
- **페이퍼 우선 8주** → 실돈 병행 시 R 0.5% 반액 → 실행 마찰 확인 후 1%.
- **12주의 정직한 목표 = 파이프라인 작동·규칙 준수율 검증이다. 엣지 판독이 아니다** `[검증 평결 반영]` — 주간 겹침+동일 매크로 창으로 유효 독립 표본이 1~2 클러스터라 40픽으로 C vs B 승률 판독은 검정력 없음. 엣지 판독은 2~3분기+, 20픽 미만 판독 금지, 60d 중간성적으로 규칙 수정 금지.
- **베토 실효성**: 관망 종목의 이후 수익 vs C — 사람 레이어의 존재 이유를 데이터로 판정(무가치로 나오면 레이어 축소). 이 프로젝트만 할 수 있는 저비용 고유 검증.
- **외부 벤치마크 추가** `[검증 평결 반영]`: 코호트 성과를 KOSPI/S&P500 동기간 수익과 병기 — "전략 전체를 접을까"의 판정 기준.
